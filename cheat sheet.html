Meta Tags

Meta tags are used to provide metadata about an HTML document, like description, keywords, and author.

            <meta name="description" content="This is a website" />
            <meta name="keywords" content="HTML, CSS, JavaScript" />
            <meta name="author" content="Mehakpreet" />
